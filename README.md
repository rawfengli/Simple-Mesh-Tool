### Simple Mesh Tool
---
这是一个简单网格生成工具，分别有简易模式和精确模式，用于可视化创建简单的网格模型。
目前主要功能是生成**2DSquare**和**3DPlane**。

简易模式下 
**Column** 表示**纵向**的顶点数 **(必须大于1)**
**Row** 表示**横向**的顶点数 **(必须大于1)**

**Length** 表示**纵向**的长度 **(上下各为为Length)**
**Width** 表示**横向**的长度 **(左右各为为Width)**

##### square

生成一张2D的四边形

##### plane

生成一张3D平面
